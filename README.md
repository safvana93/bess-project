# bess-project
Data analysis of Battery Energy Storage System(BESS)
## Project Overview
This project focuses on analyzing the performance of a Battery Energy Storage System (BESS) using Python.The main goal of project is to analyse how energy can be stored during low demand & used during peak demand to reduce electricity costs & improve efficiency.

##  Objectives
- Calculate Round Trip Efficiency (RTE)
- Analyze battery performance during charge and discharge cycles
- Study the impact of temperature on efficiency
- Evaluate cost savings using Peak Shaving

## Key Analysis

## Performance Tracking
Calculated Round Trip Efficiency (RTE) to understand energy loss.

## Thermal Analysis
Analyzed how battery temperature affects performance and efficiency.

## Economic Analysis
Studied Peak Shaving to reduce electricity cost during high demand periods.
